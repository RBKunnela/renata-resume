## Deployment

### Railway

1. Create new project on Railway
2. Connect this GitHub repository
3. Railway will auto-detect static site
4. Deploy!

### Other Platforms

- **Vercel**: `vercel deploy`
- **Netlify**: Connect repo & deploy
- **GitHub Pages**: Enable in repo settings

## Local Development

1. Clone repository
2. Open `index.html` in browser
3. No build process required!

## Files

- `index.html` - Main HTML with both EN/FI content
- `styles.css` - All styling
- `script.js` - Language toggle & interactions
- `README.md` - This file

## Language Toggle

- Click "English" / "Suomi" buttons
- Or use keyboard: `Alt+1` for EN, `Alt+2` for FI
- Preference is saved to localStorage

## Hidden AI Keywords

The page includes AI-optimized keywords that are:
- Hidden from human view (off-screen positioning)
- Readable by AI/ATS systems
- Include skills, technologies, certifications
- Boost visibility in applicant tracking systems
